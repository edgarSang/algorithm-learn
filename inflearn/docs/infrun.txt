01/27 제1-2강 Recursion의 개념과 기본 예제들
    1.무한루프에 빠지지 않으려면
        -적어도 하나의 경우에 리커젼으로 빠지지않는 루트가 존재해야한다
    2.recursion은 수학적 귀납법과 동일하다.
    -정리: func(int n)은 음이아닌 정수 n 에 대해서 0~n 합을 올바로 계산한다.
    -증명
        1. n=0 이면 n=0은 0을 반환한다 (O)
        2. 임의 양의정수 k에 대해서 n<k인 경우 0~n 합을 올바르게 계산하여 반환한다고 가정. // k보다 작은 n에대해 모두성립한다가정
        3. n=k인 경우를 고려해보자. func은 먼저 func(k-1) 호출하는데 2번의 가정에 의해서 0에서 k-1까지의 합이 올바르게 계산되어 반환.
        메서드 func은 그값에 n을 더해서 반환한다. 따라서 func은 0에서 k까지의 합을 올바로 계산하여 반환한다.
